=== STEP 3: ACTIVATE THE VAULT ===

The vault is deployed but has no position yet (positionId == 0).
To activate it, follow the same flow as the existing vaults:

3a. MINT a V4 BURGERS/FTP position NFT directly to NEW_VAULT_ADDRESS
    (or transfer an existing position using POSM.safeTransferFrom).
    Pool:       BURGERS/FTP  fee=10000 tickSpacing=200 hooks=0x0
    Range:      tickLower=416600 tickUpper=424800
    POSM:       0x58daec3116aae6D93017bAAea7749052E8a04fA7

3b. Call vault.adoptPosition(tokenId)
    - Only callable by vaultOwner.
    - Confirms the NFT is in the right pool/range.
    - Credits owner shares for the seed liquidity.
    - After this call, positionId != 0 and the vault is live.

3c. Verify by calling vault.getInfo(someAddress) â€” should not revert.

Done. The vault is now open for public deposits via deposit(usdgAmount, displayName).
